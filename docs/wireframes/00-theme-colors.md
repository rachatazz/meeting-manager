# Theme & Design System

## Color Palette

### Primary Colors

- **Primary Blue**: `#1E3A8A` - Deep blue for headers, primary buttons
- **Primary Orange**: `#F97316` - Accent color for CTAs, highlights
- **Secondary Blue**: `#3B82F6` - Links, secondary actions
- **Dark Navy**: `#0F172A` - Text, backgrounds

### Neutral Colors

- **White**: `#FFFFFF` - Backgrounds, cards
- **Light Gray**: `#F8FAFC` - Subtle backgrounds
- **Medium Gray**: `#64748B` - Secondary text
- **Border Gray**: `#E2E8F0` - Borders, dividers

### Status Colors

- **Success Green**: `#10B981` - Confirmed status
- **Warning Yellow**: `#F59E0B` - Pending status
- **Error Red**: `#EF4444` - Cancelled, errors
- **Info Blue**: `#3B82F6` - Information

### Semantic Usage

- **Confirmed**: Green (#10B981)
- **Pending**: Yellow (#F59E0B)
- **Cancelled**: Red (#EF4444)
- **Completed**: Blue (#3B82F6)

## Typography

### Font Family

- **Primary**: Inter, system-ui, sans-serif
- **Headings**: Inter, bold/semibold
- **Body**: Inter, regular/medium

### Font Sizes

- **H1**: 2.5rem (40px) - Page titles
- **H2**: 2rem (32px) - Section headers
- **H3**: 1.5rem (24px) - Card titles
- **H4**: 1.25rem (20px) - Subsections
- **Body**: 1rem (16px) - Regular text
- **Small**: 0.875rem (14px) - Meta info, labels
- **Tiny**: 0.75rem (12px) - Badges, timestamps

## Spacing System

- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)
- **2xl**: 3rem (48px)
- **3xl**: 4rem (64px)

## Border Radius

- **sm**: 0.375rem (6px) - Buttons, inputs
- **md**: 0.5rem (8px) - Cards
- **lg**: 0.75rem (12px) - Modals
- **full**: 9999px - Pills, avatars

## Shadows

- **sm**: 0 1px 2px rgba(0, 0, 0, 0.05)
- **md**: 0 4px 6px rgba(0, 0, 0, 0.1)
- **lg**: 0 10px 15px rgba(0, 0, 0, 0.1)
- **xl**: 0 20px 25px rgba(0, 0, 0, 0.15)

## Component Styles

### Buttons

- **Primary**: Orange background (#F97316), white text, rounded-md
- **Secondary**: Blue outline (#3B82F6), blue text, rounded-md
- **Danger**: Red background (#EF4444), white text, rounded-md
- **Ghost**: Transparent, gray text, hover background

### Cards

- White background, border-gray border, rounded-md, shadow-sm
- Hover: shadow-md transition

### Inputs

- Border-gray border, rounded-md, focus:border-blue
- Height: 2.5rem (40px)
- Padding: 0.5rem 0.75rem

### Badges/Tags

- Rounded-full, small text, colored background with opacity
- Padding: 0.25rem 0.75rem
